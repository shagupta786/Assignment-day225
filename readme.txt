# Internship Application Portal

## Setup

1. Import `database.sql` in phpMyAdmin.
2. Place folder in `htdocs/` in XAMPP.
3. Access via `http://localhost/internship_portal/`.
4. Default admin login: `admin / admin123`.

## Structure

- `index.html`: user registration
- `register.php`: handles form + upload
- `/uploads/`: saved resumes
- `/admin/`: admin login and dashboard
- `/includes/`: DB connection
